
# تصحيح احترام العقد البصري لتطبيق المركز الإعلامي

## المشكلة

الواجهة كانت تعمل، لكنها لا تعكس هوية PalWakf بصريًا بما يكفي.

## ما تم تصحيحه

```text
1. إنشاء visual contract خاص بتطبيق المركز الإعلامي.
2. اعتماد ألوان المنصة: dark / gold / royal red.
3. جعل زر "نشر رسمي" واضحًا كإجراء مؤسسي.
4. ضبط Bottom Navigation بعيدًا عن ألوان Material الافتراضية.
5. تحسين واجهة "الموقع الرسمي أولًا".
6. تحسين واجهة الجمهور للرابط الرسمي.
7. توضيح أن public هو API edge فقط.
```

## لا يوجد

```text
لا SQL
لا public base tables
لا service_role
لا production approval
```
