# Platform Technical Services — Evidence, Notifications & Operations Center Mega Batch

## ما أضيف

- جدول أدلة تشغيلية للطلبات التقنية.
- جدول إشعارات داخلية.
- جدول قرارات تشغيلية.
- RPCs لإضافة evidence، وتسجيل decision، وإنشاء/قراءة notification.
- توسيع dashboard RPC ليعرض evidence/notifications/operation_decisions.
- امتدادات Flutter لاستدعاء هذه العمليات.

## الحدود

لا backup/restore فعلي، لا إغلاق تلقائي، لا deploy من Flutter، لا service_role، لا تعديل بيانات سيادية، لا production approval.
