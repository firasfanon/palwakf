# تقرير شامل — مستكشف الوقف v4.0

## نظرة عامة
بوابة رسمية لاستكشاف الأوقاف الفلسطينية تاريخيًا وإداريًا، مع لوحات تحكم لإدارة المحتوى والمستخدمين.

## التقانة
Flutter Web (RTL)، Riverpod، go_router، Supabase (Postgres/PostGIS)، flutter_map.

## ميزات الإصدار v4.0
- إدارة المستخدمين بالأدوار: superuser / admin / user / viewer + أذونات تفصيلية.
- إدارة الصفحة الرئيسة (عنوان/وصف/إظهار الأقسام).
- إعدادات الموقع (اسم/لون/روابط اجتماعية/الوضع الليلي).
- صفحات أساسية: خريطة/تشريعات/حول/تواصل/تسجيل دخول.

## النشر
1) املأ .env بقيم Supabase.
2) flutter create .
3) flutter pub get
4) flutter run -d edge  أو  flutter build web

## خارطة طريق
v4.1 CRUD كامل للأراضي + مضلعات + Popup بطاقات على الخريطة.
v4.2 مكتبة وثائق تاريخية وبحث متقدم.
v4.3 تكامل الطاقة للمساجد وتقارير الترشيد.
