# PalWakf Platform 16 — Public Roadmap Candidate

This package adds the Ministry home `PalWakf` navigation entry and a responsive `/palwakf` roadmap page.

Use the `updates-only` archive to overlay the included source files on the local Flutter project. The apply script creates a timestamped backup before copying any file.

The route is Ministry-only by design. Unit public navigation remains free of central Ministry/platform-only links.
