# Governing Contract Appendix — PWF-SIS N2.62

## Adoption Program
PWF-SIS is now governed as an adoption program, not a page-by-page repair track.

## Mandatory classification
Every page/route must be classified before migration.

## High-risk lock
High-risk systems are readiness-only until system-specific contracts are written.

## No database coupling
PWF-SIS adoption must not trigger schema migration, SQL execution, or source-of-truth changes.
