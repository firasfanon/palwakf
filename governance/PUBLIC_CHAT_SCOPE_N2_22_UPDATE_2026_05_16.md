# Public Chat Scope Update — N2.22

الشات العام لا يعرض معلومات من جداول سيادية أو انتقالية مباشرة. أي معرفة عامة يجب أن تأتي من:

- published public views
- approved assistant knowledge
- public_content wrappers
- public_visibility=true where applicable

لا يستخدم public cache أو legacy tables كمصدر حقائق نهائية.
