# Governing Contract Appendix — PWF-SIS N2.66A

## Public UI repair rule
Public responsive repairs may adjust layout, padding, wrapping, alignment, and safe widget composition.

## Forbidden
- Changing public data contracts.
- Changing service submit/track behavior.
- Changing published-only filters.
- SQL execution.
- Database Wave B.
- `waqf_assets` mutation.
- Production approval.

## Evidence gate
N2.66 cannot close until public runtime browser evidence is free of RenderFlex overflow and ParentDataWidget exceptions.
